# Charles

<p align="center">
  <img src="charles_logo.png" alt="Charles Logo" width="240">
</p>

Charles is a personal agent for running errands and tasks. It's built on [Flue](https://flueframework.com/), [Cloudflare Workers](https://workers.cloudflare.com/), and [Browser Run](https://developers.cloudflare.com/browser-run/), and runs entirely on Workers.

## Contributions

Not open for contributions. Just open source. Entirely unsupported.
